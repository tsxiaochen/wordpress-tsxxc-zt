**tsxxc主题特色**

响应式布局，可兼容PC、平板、手机

完善的后台选项设置，几乎不用修改模板即可完成漂亮的布局

良好的SEO优化

文章浏览量统计

后台自带友情链接设置

Gravatar头像加速

缩略图延迟加载

文章点赞、打赏功能

彩色标签

评论显示归属地和UA

风格简约，非常适合个人博客

博客/cms布局

文章列表式/卡片式
